# Final Project - MYGRAM

Cara Menjalankan

- Buat Database
- Clone Github
- Masuk ke Folder final-project
- Ubah name .env.example ke .env dan ubah isi file .env sesuai dengan database yang digunakan
- Jalankan dengan perintah di terminal
```
go run main.go
```
